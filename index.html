CSC-20021 Web Technologies Assessment - 70%

For the first part of the coursework, your task is to create a website for various components, which are listed in Table 1. The website should be developed using HTML5 and CSS3, and should be responsive adjusting intelligently to different display capabilities. Your work will be evaluated using the latest stable release of the Google Chrome web browser.

CV - 15%

Your HTML5-based CV as a web page. Assume that this CV will be used to apply for a web developer position. Do not include personal details such as your names or photos. The page should include a profile summary and information about your computer science skills, achievements, modules and the areas in which you would like to do your third-year projects. Module names should appear as links to the Keele website, and links should be working. Please be creative and extend this page with more details.


Weather Data - 15%

A web page to include the following tasks from Practical 4:
i. Practical 4 - Task 2 - [10%]: Using Chart.js library to visualise weather forecast information.
ii. Practical 4- Task 3 - [5%]: Customise the appearance of the graph from Practical 4 - Task 2.


Login System - 10%

Administrators can login and logout, and the authentication details are kept using the PHP session. Administrator details must be stored in a file and should be validated. Please DO NOT include the functionality to add new administrators.


Other Components

The current time widget in the top right is a simple JavaScript program that gets the local time.
